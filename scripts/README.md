# 資料庫初始化腳本說明

## init_db.py - 整合初始化腳本

這是一個整合的資料庫初始化腳本，可以：
1. 從 `knowledge/truth_law.json` 初始化法條資料
2. 使用 LLM 預先生成選擇題和簡答題各 40 題

### 使用方式

#### 基本用法

```bash
# 初始化本地資料庫（法條 + 題目）
python scripts/init_db.py --local

# 初始化遠端資料庫（法條 + 題目）
python scripts/init_db.py --remote

# 同時初始化本地和遠端（法條 + 題目）
python scripts/init_db.py --both
```

#### 進階選項

```bash
# 只初始化法條，跳過題目生成
python scripts/init_db.py --local --skip-questions

# 只生成題目，跳過法條初始化（假設法條已存在）
python scripts/init_db.py --local --skip-laws

# 例如：如果法條已經存在，只需要補充題目
python scripts/init_db.py --both --skip-laws
```

### 題目生成邏輯

- **隨機選擇法條**：使用 `random.sample()` 打亂法條順序，確保題目分佈多樣化
- **避免重複**：生成前會檢查該法條的最近 3 題，避免產生相似問題
- **平衡分佈**：選擇題 40 題 + 簡答題 40 題，分散在不同法條上

### 注意事項

1. **需要 API Key**：題目生成需要 `OPENROUTER_API_KEY` 環境變數
2. **需要連線**：確保可以連接到 MongoDB（本地或遠端）
3. **執行時間**：生成 80 題大約需要 10-20 分鐘（取決於 API 響應速度）

### 環境變數

在 `.env` 文件中設置：

```env
# MongoDB 連線（本地）
MONGO_URI=mongodb://localhost:27017/localdb

# OpenRouter API Key（用於題目生成）
OPENROUTER_API_KEY=your_api_key_here
```
